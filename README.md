# PDF印刷用まとめツール

複数のPDFを順番調整しながら1つに結合し、両面印刷しやすい形に整えるためのFlask製ツールです。

## 作品紹介文

複数のPDFを、印刷しやすい順番に並べ替えて、そのまま1つに結合できるツールです。奇数ページのPDFが混ざっていても、次の資料が裏面始まりにならないよう空白ページの挿入有無を確認しながら、印刷向けの形に整えられます。

単なるPDF結合ではなく、アップロード、順番調整、不要ファイルの削除、奇数ページの確認、出力ファイル名の指定まで、実際の作業フローに合わせて使えるように作っています。

会議資料、配布資料、授業プリント、提出用PDFなど、複数ファイルをまとめて両面印刷したい場面で使いやすい、小さな実務ツールです。

# PDF印刷用まとめツール

複数のPDFを順番調整しながら1つに結合し、両面印刷しやすい形に整えるためのFlask製ツールです。

## できること

- 複数のPDFをアップロードする。
- アップロードしたPDFの順番を上下ボタンで並び替える。
- 不要なPDFを個別に削除する。
- 奇数ページのPDFを確認する。
- 必要に応じて、次の資料を表面から始めやすくするための空白ページ挿入を反映する。
- 結合後のPDFファイル名を指定してダウンロードする。
- 同名ファイルをアップロードした時に、自動で別名保存する。

## 想定している使い方

- 会議資料をひとつにまとめて印刷したい時
- 配布資料を順番どおりに結合したい時
- 奇数ページの資料が混ざっていて、両面印刷時の裏面始まりを避けたい時
- 提出用PDFをまとめたい時

## 画面の流れ

1. PDFを複数アップロードする。
2. 一覧画面で順番を調整し、不要なものを削除する。
3. 必要なら空白ページ自動挿入をONにする。
4. 確認画面で奇数ページPDFや想定ページ数を確認する。
5. 結合PDFを指定したファイル名でダウンロードする。

## 技術構成

- Python
- Flask
- PyPDF2
- HTML / CSS / JavaScript

## 特徴

このツールは、単にPDFを結合するだけではなく、印刷前の確認作業まで含めて扱えるように設計しています。

特に、奇数ページPDFの確認と空白ページ挿入の考え方をUI上で見えるようにしているため、両面印刷を前提にした資料づくりに向いています。

## 今後の改善候補

- 結合後に元PDFをどう扱うかの選択肢追加。
- 例外時のエラーメッセージ改善。
- 画面文言や空状態のさらに細かい調整。

## 起動方法

python app.py
